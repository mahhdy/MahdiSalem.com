# Contributing / راهنمای مشارکت

Thank you for your interest in contributing! This project welcomes community input through several channels.

از علاقه شما به مشارکت سپاسگزاریم! این پروژه از مشارکت جامعه از طریق چندین کانال استقبال می‌کند.

## How to Contribute / نحوه مشارکت

### 1. Suggest Edits to Books / پیشنهاد ویرایش کتاب‌ها

If you find errors or have suggestions for book content:

1. Navigate to the chapter you want to edit
2. Click the "Suggest Edit" button
3. Fill in the GitHub Issue template with your suggestion
4. The maintainer will review and potentially incorporate your changes

### 2. Suggest Wiki Amendments / پیشنهاد اصلاح ویکی

For collaborative documents like the Proposed Constitution:

1. Go to the wiki page
2. Click "Suggest Amendment"
3. Describe your proposed change clearly
4. Include reasoning for the amendment

### 3. Report Issues / گزارش مشکلات

- Use [GitHub Issues](https://github.com/YOUR_USERNAME/mahdisalem.com/issues) to report bugs or suggest features
- Please check existing issues before creating a new one

### 4. Translations / ترجمه

Help translate content between Persian and English:

1. Check if the content already exists in the target language
2. Create a new file in the appropriate language directory
3. Submit a Pull Request

## Content Guidelines / راهنمای محتوا

- All content should be respectful and constructive
- Academic rigor is expected for philosophical and policy content
- Sources should be cited where applicable
- Content should align with the mission of promoting democracy and human rights

## Code Contributions / مشارکت در کد

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test locally with `npm run build`
5. Submit a Pull Request

## Code of Conduct / آیین‌نامه رفتاری

- Be respectful and inclusive
- Focus on constructive criticism
- No hate speech or personal attacks
- Disagreements should be handled with civility
