# تشریح نظام حاکم: آناتومی قدرت

::: kholasebox
جمهوری اسلامی نظامی «شبه‌تمامیت‌خواه» است که بر چهار ستون اصلی استوار است: دستگاه نظامی-امنیتی (به‌ویژه سپاه)، شبکه روحانیت و نهادهای مذهبی، اقتصاد رانتی و شبکه‌های وابستگی، و نیروهای نیابتی منطقه‌ای. این نظام با بحران‌های متعدد مشروعیت، کارآمدی و جانشینی مواجه است. شناخت دقیق ساختار قدرت، نقاط ضعف و شکاف‌های درونی آن، پیش‌نیاز هر استراتژی موفق گذار است.
:::

### ایدئولوژی: ولایت فقیه

::: naghlbox
«ولایت فقیه» ستون فکری نظام است: حاکمیت الهی از طریق فقیه جامع‌الشرایط. این نظریه:

- مشروعیت را آسمانی و نه زمینی (مردمی) می‌داند

- رهبر را پاسخگوی خدا می‌داند، نه مردم

- نقد رهبری را کفر و خروج از دین تلقی می‌کند

- مرزهای قدرت رهبری را نامحدود می‌داند
:::

::: hashdarbox
ایدئولوژی ولایت فقیه دچار بحران عمیق شده است:

- اکثریت مراجع تقلید شیعه آن را قبول ندارند

- نسل جوان هیچ تعلقی به آن ندارد

- حتی بخش‌هایی از روحانیت حکومتی زیر سؤالش می‌برند

- عملکرد ۴۵ ساله نظام، وعده‌های ایدئولوژیک را نقض کرده
:::

::: strategybox
سپاه فقط یک نیروی نظامی نیست؛ یک امپراتوری اقتصادی است:

- **قرارگاه خاتم‌الانبیا:** بزرگ‌ترین پیمانکار کشور (نفت، گاز، زیرساخت)

- **کنترل بنادر و گمرکات:** قاچاق و واردات غیررسمی

- **بنیادهای وابسته:** میلیاردها دلار دارایی

- **سهم از اقتصاد:** تخمین ۲۰-۴۰٪ GDP

این قدرت اقتصادی، انگیزه قوی برای حفظ وضع موجود ایجاد می‌کند.
:::

## ظرفیت سرکوب: ارزیابی واقع‌بینانه

### آنچه دی‌ماه ۱۴۰۴ نشان داد

::: center
    [**مشاهده**]{style="color: white"} [**تحلیل**]{style="color: white"}
  ------------------------------------ --------------------------------------------------------------------
                        **سرعت واکنش** نظام در عرض ساعات اولیه توانست نیروهای سرکوب را بسیج کند
                         **شدت خشونت** استفاده از گلوله جنگی از روز اول؛ الگوی «جنگ شهری» نه «کنترل شورش»
                           **گستردگی** سرکوب همزمان در ۴۰۰+ شهر نشان‌دهنده لجستیک قوی
                            **آمادگی** به نظر می‌رسد نظام از قبل برای چنین سناریویی آماده بود
                          **یکپارچگی** شواهدی از نافرمانی گسترده در نیروها دیده نشد
:::

### محدودیت‌های ظرفیت سرکوب

::: olgoobox
اما سرکوب نامحدود نیست:

1.  **نیروی انسانی:** هسته قابل اعتماد محدود است (۱۰-۲۰ هزار نفر)

2.  **جغرافیا:** نمی‌توان همه‌جا را همزمان پوشش داد

3.  **زمان:** سرکوب طولانی‌مدت خسته‌کننده و فرساینده است

4.  **روحیه:** حتی نیروهای وفادار در برابر کشتار مستمر دچار فرسایش می‌شوند

5.  **هزینه بین‌المللی:** سرکوب خونین، فشار خارجی را افزایش می‌دهد

6.  **اقتصاد:** اعتصاب گسترده می‌تواند اقتصاد را فلج کند
:::

### بحران جانشینی

::: hashdarbox
خامنه‌ای ۸۵ ساله است و سؤال جانشینی حیاتی است:

- **گزینه‌ها محدودند:** مجتبی خامنه‌ای (بی‌تجربه، منفور)، رئیسی (فوت شده)، چند نفر دیگر

- **شورای رهبری:** احتمال رهبری جمعی؟ (ضعف ذاتی)

- **تنش جناحی:** هر گزینه‌ای مخالفانی دارد

::: olgoobox
نظام‌های اقتدارگرا معمولاً به یکی از این شکل‌ها سقوط می‌کنند:

1.  **فشار از پایین:** انقلاب مردمی (ایران ۵۷، رومانی ۸۹)

2.  **شکاف از بالا:** کودتا یا انشقاق نخبگان (پرتغال ۷۴)

3.  **فروپاشی تدریجی:** ناکارآمدی مزمن (شوروی ۹۱)

4.  **مذاکره:** گذار مصالحه‌ای (لهستان ۸۹، آفریقای جنوبی ۹۴)

برای ایران، سناریوی «ترکیبی» محتمل‌ترین است: فشار مردمی + بحران جانشینی + فشار خارجی.
:::

## جمع‌بندی فصل

::: strategybox
1.  **سپاه کلیدی است:** بدون ایجاد شکاف یا بی‌طرف کردن بخشی از سپاه، سرنگونی بسیار دشوار است.

2.  **اقتصاد نقطه ضعف است:** فشار اقتصادی (تحریم + اعتصاب) می‌تواند نظام را فلج کند.

3.  **جانشینی فرصت است:** آماده باشید برای بهره‌برداری از بحران جانشینی.

4.  **شکاف‌ها را تقویت کنید:** پیام‌های متفاوت به جناح‌های مختلف.

5.  **واقع‌بین باشید:** نظام هنوز توان سرکوب دارد. برنامه‌ریزی بلندمدت لازم است.
:::

## پرسش‌هایی برای تأمل {#پرسشهایی-برای-تأمل .unnumbered}

1.  کدام ستون قدرت نظام به نظر شما آسیب‌پذیرترین است؟ چرا؟

2.  آیا فکر می‌کنید ایجاد شکاف در سپاه ممکن است؟ چگونه؟

3.  بحران جانشینی چه فرصت‌هایی برای اپوزیسیون ایجاد می‌کند؟

4.  تجربه دی‌ماه چه درس‌هایی درباره ظرفیت سرکوب نظام داد؟

5.  کدام سناریوی فروپاشی را محتمل‌تر می‌دانید؟

## منابع این فصل {#منابع-این-فصل .unnumbered}

- تحلیل‌های مراکز پژوهشی (IISS، Crisis Group، و غیره)

- گزارش‌های رسانه‌های مستقل فارسی‌زبان

- پژوهش‌های دانشگاهی درباره ساختار قدرت ایران

- شهادت‌های افراد مطلع (خارج‌شده از نظام)

- تفصیل بیشتر در پیوست ج (ساختار قدرت ج.ا.)
