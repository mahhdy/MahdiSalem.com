# **تحلیل جامع انقلاب آمریکا (۱۷۶۵-۱۷۸۳) و تأثیر آن بر انقلاب فرانسه**

---

## **خلاصه‌ی اجرایی**

انقلاب آمریکا **نخستین انقلاب موفق دوران مدرن** است که منجر به استقلال سیزده مستعمره‌ی بریتانیایی و تأسیس ایالات متحده شد. این انقلاب هم **جنگ استقلال** بود و هم **انقلاب سیاسی-ایدئولوژیک**.

| **بُعد** | **توضیح** |
| --- | --- |
| **ماهیت** | استقلال‌طلبی + جمهوری‌خواهی |
| **مدت** | ۱۷۶۵-۱۷۸۳ (۱۸ سال) |
| **جنگ** | ۱۷۷۵-۱۷۸۳ (۸ سال) |
| **نتیجه** | استقلال، جمهوری فدرال |
| **تلفات** | ~۲۵,۰۰۰ آمریکایی، ~۲۴,۰۰۰ بریتانیایی |
| **تأثیر بر فرانسه** | ایدئولوژیک + مالی + شخصی |

---

## **طرح کلی تحولات**

```
text
۱۷۶۳: پایان جنگ هفت‌ساله
    ↓
۱۷۶۵-۱۷۷۳: بحران مالیاتی و اعتراضات
    ↓
۱۷۷۴: قوانین تنبیهی، کنگره‌ی قاره‌ای اول
    ↓
۱۷۷۵: آغاز جنگ (لکسینگتون و کانکورد)
    ↓
۱۷۷۶: اعلامیه‌ی استقلال
    ↓
۱۷۷۷-۱۷۷۸: ساراتوگا، اتحاد با فرانسه
    ↓
۱۷۸۱: پیروزی یورک‌تاون
    ↓
۱۷۸۳: معاهده‌ی پاریس (استقلال)
    ↓
۱۷۸۷: قانون اساسی
    ↓
۱۷۸۹: واشنگتن اولین رئیس جمهور
```

## **۱.۲ جنگ هفت‌ساله و پیامدهای آن (۱۷۵۶-۱۷۶۳)**

### **جنگ فرانسه و سرخپوستان**

| **طرف** | **متحدان** |
| --- | --- |
| بریتانیا | مستعمرات آمریکا، برخی قبایل |

- فرانسه کانادا و سرزمین‌های شرق می‌سی‌سی‌پی را از دست داد
- بریتانیا قدرت برتر آمریکای شمالی شد

**استدلال بریتانیا:** مستعمرات باید در هزینه‌ی دفاع سهیم شوند.

## **۲.۳ اعلامیه‌ی استقلال (۴ ژوئیه ۱۷۷۶)**

### **زمینه**

| **تاریخ** | **رویداد** |

| ژانویه ۱۷۷۶ | انتشار «عقل سلیم» (توماس پین) |
| ۷ ژوئن ۱۷۷۶ | قطعنامه‌ی لی برای استقلال |
| ۱۱-۲۸ ژوئن | نگارش اعلامیه توسط جفرسون |
| **۴ ژوئیه ۱۷۷۶** | **تصویب اعلامیه** |

### **«عقل سلیم» (Common Sense) - توماس پین**

> «حکومت، حتی در بهترین حالت، شر ضروری است؛ در بدترین حالت، شر غیرقابل تحمل.»
> 

**تأثیر:**

- ۵۰۰,۰۰۰ نسخه فروش (در جمعیت ۲.۵ میلیونی)
- تبدیل افکار عمومی به سمت استقلال
- حمله به سلطنت، نه فقط پارلمان

### **متن اعلامیه‌ی استقلال**

**نویسنده‌ی اصلی:** توماس جفرسون

**ساختار:**

| **بخش** | **محتوا** |
| --- | --- |
| **مقدمه** | فلسفه‌ی حقوق طبیعی |
| **فهرست شکایات** | ۲۷ اتهام علیه جورج سوم |
| **اعلام استقلال** | قطع رابطه با بریتانیا |

### **اصول کلیدی**

> «ما این حقایق را بدیهی می‌دانیم که همه‌ی انسان‌ها برابر آفریده شده‌اند، که خالق آنان را به حقوق مسلمی مجهز کرده است، از جمله حق زندگی، آزادی و جستجوی خوشبختی.»
> 

| **اصل** | **منبع فلسفی** |
| --- | --- |
| حقوق طبیعی | جان لاک |
| برابری ذاتی | روشنگری |
| رضایت حکومت‌شوندگان | قرارداد اجتماعی |
| حق انقلاب | لاک، سیدنی |

### **امضاکنندگان**

- ۵۶ نماینده از ۱۳ مستعمره
- **معروف‌ترین:** جان هنکاک (امضای بزرگ)
- **بنیان‌گذاران کلیدی:** جفرسون، آدامز، فرانکلین

| --- | --- |

### **لافایت: پل بین دو انقلاب**

```
text
۱۷۷۷: ورود به آمریکا (۱۹ ساله)
        ↓
۱۷۷۷-۸۱: جنگ در کنار واشنگتن
        ↓
۱۷۸۱: پیروزی یورک‌تاون
        ↓
۱۷۸۲: بازگشت به فرانسه به عنوان قهرمان
        ↓
۱۷۸۹: انقلاب فرانسه
   • فرمانده گارد ملی پاریس
   • پیش‌نویس اعلامیه‌ی حقوق بشر (با کمک جفرسون)
   • میانه‌رو (سلطنت مشروطه)
        ↓
۱۷۹۲: فرار از فرانسه (رادیکالیسم)
        ↓
زندان اتریش (۵ سال)
```

## **درس‌های کلیدی**

| **درس** | **توضیح** |
| --- | --- |
| **انقلاب ممکن است** | آمریکا ثابت کرد، فرانسه تأیید کرد |
| **زمینه مهم است** | جامعه‌ی آمریکا آماده‌تر بود |
| **رهبری کلیدی است** | واشنگتن vs روبسپیر |
| **میانه‌روی سخت است** | لافایت شکست خورد |
| **نهادسازی زمان می‌برد** | آمریکا هم بحران داشت |
| **ایده‌ها سفر می‌کنند** | اما در هر زمینه متفاوت عمل می‌کنند |
